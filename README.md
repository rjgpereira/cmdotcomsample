Instructions:

1. Change the package name to match your firebase messaging config;  
2. Replace the file "app/google-services.json" with the one downloaded from your firebase console;
3. Replace the "API_KEY" in "MainActivity.kt" with your CM.com api KEY;

**IMPORTANT**:
This sample project does not request the user permission to receive push notifications, so **MAKE SURE** you enable it yourself manually in Settings->Applications->CMdotComSample
